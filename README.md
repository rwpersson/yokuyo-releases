# Yokuyo 抑揚

### ▶ [**Download Yokuyo**](https://rwpersson.github.io/yokuyo-releases/)

A sentence-level Japanese pitch-accent visualizer for Windows — see the continuous pitch (F0) contour of any sentence, hear it, and learn the rules.

## Install

1. **Download & run the installer** — the button above, or from [**Releases**](https://github.com/rwpersson/yokuyo-releases/releases/latest). Windows may warn it's from an "unknown publisher" → click **More info → Run anyway**. *(Yokuyo isn't code-signed yet; the installer and source are open to inspect.)*
2. **First launch sets up the speech engine.** Yokuyo uses [VOICEVOX](https://voicevox.hiroshiba.jp) (free) for audio and pitch. It's detected and started for you automatically — if it isn't installed, Yokuyo opens the download page and picks it up once installed, no restart needed.
3. **Analyze** a Japanese sentence to see its pitch contour, hear it, and explore the pitch-accent rules.
4. *(Optional)* add the [**Chrome extension**](https://chrome.google.com/webstore/detail/nhjagpdjihmjoendmbmebnhjeedfbhpb) to use Yokuyo while you browse — hold **Shift** and hover Japanese text on any page (works while the desktop app is running).

**Requires Windows 10/11.** No audio? Make sure VOICEVOX is running (Yokuyo starts it for you).
